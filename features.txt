CodeMaker Requirements:

CodeMaker is an IDE that allows users to visualize, refactor, and develop their projects more easily.

IDE

- CodeMaker will contain a basic text editor
-- The text editor will read a file and display its contents in an editable control
-- The text editor will save its contents to a file (overwrite or new file)

- CodeMaker will display the file structure of a project
-- The file structure display will show all files and subfolders of a selected folder
-- The file structure display will allow users to quickly select and open a file for editing

Visualize

- CodeMaker will lint C# code
-- The linter will highlight keywords
-- The linter will highlight literals
